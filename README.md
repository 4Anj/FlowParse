## Getting Started with FlowParse
Prerequisites
```bash
Python 3.10+
Node.js 18+
npm
```

## Install dependencies in the project folder
BACKEND:
```bash
pip install scapy numpy pandas scikit-learn torch tensorflow matplotlib seaborn
```
FRONTEND:
```bash
cd flow-parse
npm install
npm start
```

## Run the React project

In the project directory, you can run:
```bash
npm start
```

Open link to view it in your browser.
```bash
http://localhost:3000
```
